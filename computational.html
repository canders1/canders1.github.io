Computational Modeling Work

Bayesian Modeling of Perspective

I work on using Bayesian inference to model pragmatics in conversation. In particular, I've developed a perspectival version of the Rational Speech Acts model to capture how conversation participants interpret perspectival items in different utterance contexts. This work was implemented in the WebPPL probabilistic programming language.

Experimental Work

Temporal Reference in English

I am investigating shifted readings of temporal adverbials like "tomorrow" in English. Ordinarily, these expressions use utterance time as their reference point, but in shifted readings, they anchor instead to a salient reference time in the previous context. I'm running experiments using Lego comic strips to explore how speakers interpret these shifted readings.
